My current project A 3D printed robot arm with 6 degrees of freedom for task automation. I started this project because of the Arduino Braccio, I thought I could make one without a kit, with more capabilities! I did!, since then this project has gotten far more complex than I planned and I am now working on adding Teleoperation, Object Detection, and a Lidar system. I used the design from the open-source PAROL6 robot from Source Robotics The robot was 3d printed using PETG, with the vision capabilities coming from the use of ROS and OpenCV. more updates and pictures on building and testing code soon!
